# 🏥 KidsCare Clinic - Centro de Neurodiversidade

O **KidsCare Clinic** é uma plataforma completa de gerenciamento para clínicas de atendimento infantil especializadas em **neurodiversidade**. O software foi desenvolvido para facilitar a conexão entre pacientes (crianças e seus responsáveis), profissionais da saúde e a administração da clínica, garantindo um cuidado centrado e humanizado.

---

## 🚀 Principais Funcionalidades

### 👨‍👩‍👧‍👦 Portal do Paciente & Responsável
- **Cadastro Simples:** Registro de responsáveis e dependentes.
- **Agendamento Online:** Marcação de consultas e sessões de terapia de forma intuitiva.
- **Histórico de Atendimento:** Visualização de consultas passadas e futuras.

### 🩺 Área do Profissional (Médicos e Terapeutas)
- **Agenda Médica:** Gestão completa de horários e disponibilidade.
- **Acompanhamento:** Visualização de consultas agendadas e prontuários básicos.
- **Controle de Acesso:** Login seguro via PIN ou senha individual.

### ⚙️ Painel Administrativo
- **Gestão de Equipe:** Cadastro, edição e remoção de profissionais da clínica.
- **Controle Total:** Visão geral da operação da clínica e fluxos de agendamento.
- **Segurança:** Autenticação administrativa robusta para proteção de dados sensíveis.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza uma stack moderna para garantir performance e escalabilidade:

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript.
- **Backend:** Python com **Flask**.
- **Banco de Dados:** **Supabase** (PostgreSQL) para armazenamento persistente e seguro.
- **Segurança:** **Bcrypt** para criptografia de senhas e proteção de sessões.
- **Design:** Focado em acessibilidade e identidade visual acolhedora para o público infantil.

---

## 🎨 Diferenciais do Projeto
Este software não é apenas um sistema de agendamento comum; ele foi projetado considerando as necessidades específicas de clínicas multidisciplinares (Psicologia, Fonoaudiologia, Terapia Ocupacional, etc.), oferecendo uma interface limpa e profissional tanto para os pais quanto para a equipe técnica.
